# WeldChecker

Это небольшое веб-приложение для обнаружение дефектов сварочных швов и предоставления пользователю подробной информации о дефектах

## Запуск в Dev режиме
```shell
yarn install
yarn dev
```

## Запуск в Preview режиме
```shell
yarn install
yarn build
yarn preview
```

## Билд статики
```shell
yarn install
yarn build
```

Статические файлы будут лежать в ./dist
